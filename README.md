# Kenya Labour Party - Official Website

A modern, responsive landing page for the Kenya Labour Party built with Next.js 14, Tailwind CSS, and GSAP animations.

## Features

- 🎨 Modern, sleek design with Kenyan national colors
- 📱 Fully responsive (mobile, tablet, desktop)
- ⚡ Fast loading with Next.js App Router
- 🎭 Smooth GSAP animations
- ♿ Accessible and SEO-optimized
- 🎯 Key sections: Manifesto, Leadership, News, Events, Membership

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

```bash
npm run build
npm start
```

## Technology Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** GSAP
- **Language:** TypeScript
- **Fonts:** Inter & Roboto
